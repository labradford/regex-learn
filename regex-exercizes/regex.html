Regex Examples

Thanks for coming to Regex Demystified. Let's see some examples of regular expressions.

Literal (Exact) Match

"Technology is a useful servant but a dangerous master."
search for exact match on the word dangerous

let pattern = /dangerous/

Result:

Case insensitive

"Technology is a useful servant but a dangerous master."
search for match on the word technology - case insensitive

let pattern = /technology/i

Result:

Global

"Even when you take a holiday from technology, technology doesn't take a break from you"
search for match on the word technology - case insensitive and global

let pattern = /technology/gi

Result:

Brackets

"A computer once beat me at chess, but it was no match for me at kick boxing"
return a count of all of the vowels

let pattern = /[aeiou]/ig

Result:

Brackets (with options)

"pat pet pit pot put"
return all of the words that start with p, end with t and contain either an a e or u

let pattern = /p[aeu]t/g

Result:

Brackets (numbers)

"Here are the three types: pie, pi, and 3.14"
return the pie numbers

let pattern = /[0-9]/g

Result:

Starts with ^

"Bond. James Bond."
checks string to see if it starts with Bond

let pattern = /^bond/i

Result:

Starts with ^

"Bond. James Bond."
checks string to see if it starts with James

let pattern = /^james/i

Result:

Find both ( | )

"Bond. James Bond."
checks string to see if it has either James or Bond

let pattern = /(james|bond)/i

Result:

Word Characters

"Always give 100%!"
search for word characters

let pattern = /\w/g

Result:

Non-word Characters

"Always give 100%!"
search for non-word characters

let pattern = /\W/g

Result:

Whitespace Characters

" Technology doesn't address everything - for example, air travel still sucks. "
find the count of whitespace (also includes carriage return, tab, form feed, and new line characters)

let pattern = /\s/

Result:

Anchor character $

"The last car on a train is the caboose"
search the end of the string

let pattern = /caboose$/

Result:

Find 0 or more

"To err is human, but to really foul things up you need a computer"
search for 0 or more (you don't want a missing character to prevent a match)

let pattern = /to*/ig

Result:

Find 1 or more

"Technology has become as ubiquitous as the air we breathe, so we are no longer conscious of its presence"
search for 1 or more

let pattern = /as+/g

Result:

you use the plus sign + to look for one or more characters and the asterisk * to look for zero or more characters. But sometimes you want to match a certain range of patterns. This is how you can specify a range.

Find a Range

"Ohhhh No!!!"
find between 3 and 6

let pattern = /oh{3,6}/i

Result:

Range w/Lower

"Helloooooooo World!"
find at least 2

let pattern = /hello{2,}/i

Result:

Exact Range

"Are you reaaaadddddy?"
find exact count

let pattern = /rea{4}d{5}y/i

Result:

Optional Character

You can specify the possible existence of an element with a question mark, ?. This checks for zero or one of the preceding element. You can think of this symbol as saying the previous element is optional.

"The color of the colonel's uniform was a blue colour"
find alternate spellings of color

let pattern = /colou?r/gi

Result:

Lookaheads

Lookaheads are patterns that will look-ahead in your string to check for patterns further along. This is useful when you want to search for multiple patterns over the same string. There are both positive (?=) and negative (?!) lookaheads. The positive lookahead sees if something is a match. The negative lookahead sees if something does not match. Here are some examples of how you might use a positive lookahead.

"happy1"
The password must contain from 4 to 8 characters, and include at least one number.

let pattern = /(?=\w{4,8})(?=\D*\d)/

Result:
"password123"
The password must be at least 5 characters long, and include two consecutive digits.

let pattern = /(?=\D*\d{2})(?=\w{5,})/

Result:

So now we have an idea on how to create regular expressions and how to match and test them. Searching can be very useful, but it is even better to use the match to find and replace items in your code

Find and Replace

"The sky is gray"
Find the word gray and replace it with blue.

let pattern = /gray/i

The sky is gray
